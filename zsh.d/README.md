README
======
私的 zsh 設定ファイルを管理する場所。

想定環境
--------
* OS
    * ubuntu 
    * MAC
    * その他

ブランチの説明と各OSの対応
------------------------------
* master
    * OSに関係なく共通で利用できるので以下の通りご利用ください。


利用にあたり必要なパッケージ
---------------------------
* ubuntu の場合は以下のパッケージが必要になりるので、パッケージ管理で導入してくてください。
    * xclip
        * ターミナルとブラウザなどでクリップボードが共有できるようになります。
